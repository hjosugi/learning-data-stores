# SQLite Workload Lab

A Dockerless data-store exercise using Python's built-in SQLite driver.

It models three workloads:

- relational task lookup
- time-series event aggregation
- graph-like edge traversal

## Run

```bash
python3 projects/sqlite-workload-lab/app.py
```

## Exercise

Move one workload to DuckDB, Kuzu, or a vector database and document what became easier or harder.
