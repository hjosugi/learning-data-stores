# DuckDB Analytics Lab

Hands-on DuckDB project for local analytical SQL over generated event data.

## Run With uv

```bash
uv run --with duckdb python projects/duckdb-analytics-lab/app.py
```

## Environment Check

```bash
python3 projects/duckdb-analytics-lab/check_duckdb.py
```

## What To Learn

- in-process analytical database workflow
- SQL aggregation over event-shaped data
- when DuckDB is a better fit than a transactional database
