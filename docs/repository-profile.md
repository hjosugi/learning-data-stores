# Repository Profile: learning-data-stores

Last verified: 2026-06-20

## GitHub Description

Database learning repo for DuckDB, H2, InfluxDB, LanceDB, Chroma, Kuzu, Neo4j, and data-store tradeoffs.

## Topics

- `learning`
- `database`
- `duckdb`
- `influxdb`
- `vectordb`
- `neo4j`

## Audience

Engineers who want to compare databases by workload, query shape, embedded/server deployment model, and operational tradeoffs.

## First Milestones

1. Add DuckDB analytics and time-series-shaped SQL examples.
2. Add H2 embedded relational example.
3. Add LanceDB vector search example.
4. Add Chroma local retrieval example.
5. Add InfluxDB 3 Core local process example.
6. Add Kuzu and Neo4j graph comparison examples.

## Public Safety Checklist

- Generated sample data only.
- No production database dumps.
- No cloud credentials or tokens.
- Every local data directory is ignored by Git.
- Every example documents how to remove generated data.
